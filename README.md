# KudiWise Backend

KudiWise is a user-friendly crypto payment and investment app targeting Zongo and Asante communities.  
Users can learn crypto trading in Hausa or Asante Twi via an AI tutor.

## Features
- User authentication
- Wallets & transactions
- Buy/sell crypto
- AI Tutor lessons
- Admin dashboard

## Tech Stack
- Python 3.11
- Django 5.x
- Django REST Framework
- PostgreSQL (planned)
